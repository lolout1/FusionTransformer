\section{Results}
\label{sec:Results}

We present comprehensive experimental results evaluating the impact of Kalman fusion on fall detection performance. All experiments use 21-fold Leave-One-Subject-Out Cross-Validation (LOSO-CV) with young subjects as test folds and elderly subjects contributing ADL samples for training. We report F1 score as the primary metric given the class imbalance inherent in fall detection (falls are rare events). Each experiment was conducted with a fixed random seed for reproducibility.

\label{sec:kalman_ablation}

The only variable changed was whether the input was processed through our linear Kalman filter to extract orientation features (roll, pitch, yaw) or used as raw accelerometer and gyroscope signals.

The orientation representation provides a more interpretable and physically meaningful feature space for fall detection, as falls are characterized by specific orientation changes rather than raw angular velocities.

\subsubsection{Why Kalman Fusion Helps Transformers}

Kalman fusion transforms raw gyroscope angular velocity ($\omega_x, \omega_y, \omega_z$ in rad/s) into orientation angles (roll, pitch, yaw in radians) by fusing accelerometer and gyroscope measurements. The transformer's self-attention mechanism computes global temporal dependencies through:
\begin{equation}
\text{Attention}(Q,K,V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
\end{equation}
Raw angular velocity is difficult to interpret---the same $\omega_z$ value could mean standing still with slight drift or mid-fall rotation. Kalman-fused orientation provides \textit{absolute} body pose information: a roll angle of $-\pi/2$ unambiguously indicates a side-lying position regardless of how fast the rotation occurred. This semantic clarity enables transformers to learn meaningful temporal attention patterns over body orientation changes during falls.

\subsubsection{Why Kalman Fusion Hurts CNNs}

CNN convolutions excel at detecting local patterns through learned kernels. Fall detection with CNNs relies heavily on detecting \textit{magnitude spikes}---sudden high-amplitude changes in angular velocity that accompany impact. The raw gyroscope captures these transient events directly. However, Kalman fusion converts velocity to position (orientation), transforming sharp spikes into gradual angle changes. A fall that produces a distinctive 500$^\circ$/s spike in raw gyroscope becomes a smooth 90$^\circ$ orientation change after fusion. The CNN kernels, optimized for detecting local discontinuities, find less distinctive patterns in the fused representation.

\subsection{Loss Function Analysis}
\label{sec:loss_analysis}

Given the severe class imbalance in fall detection (falls represent a minority of samples), we investigated the impact of focal loss versus binary cross-entropy (BCE) loss.

\centering

\label{tab:loss_comparison}

\midrule
BCE & 88.01 $\pm$ 8.4 & 84.26 $\pm$ 10.0 & \textbf{92.17 $\pm$ 7.5} & 85.70 $\pm$ 13.6 \\

\midrule

}
\vspace{1mm}
\scriptsize
\textit{Focal loss: $\alpha=0.75$, $\gamma=2.0$}
\end{table}

Table~\ref{tab:loss_comparison} reveals that focal loss improves F1 score by +0.85\% primarily through a substantial increase in recall (+6.41\%). This is clinically significant for fall detection systems, where missing a fall (false negative) has more severe consequences than a false alarm. The focal loss down-weights easy negatives (ADL samples correctly classified with high confidence), allowing the model to focus learning on the harder fall detection task. While precision decreases with focal loss, the overall F1 score improves due to the recall gains.

The improvement stems from two factors: (1) Kalman fusion provides semantically meaningful orientation features, and (2) separate processing of accelerometer and orientation streams allows the model to learn modality-specific patterns before fusion. Note that adding SE to single-stream raw input actually slightly decreased performance (87.80\% vs 86.99\%), suggesting that channel attention is less beneficial when processing heterogeneous raw signals.

\subsection{Summary of Key Findings}

Our experimental evaluation leads to the following conclusions:

\begin{enumerate}
    \item \textbf{Kalman fusion improves transformer-based fall detection}: Under controlled conditions with identical architecture, Kalman-fused orientation features outperform raw gyroscope data by +1.40\% F1 score (90.89\% vs 89.49\%) for transformers.

    \item \textbf{SE and TAP synergy with Kalman fusion}: The combination of SE attention and TAP pooling provides the greatest benefit when used with Kalman fusion (+1.77\% over raw with SE+TAP: 90.85\% vs 89.08\%).

    \item \textbf{Dual-stream architecture is superior}: The dual-stream transformer (86.99\% $\rightarrow$ 90.85\%) significantly outperforms single-stream alternatives (+3.86\%), validating separate processing of accelerometer and orientation streams.

    \item \textbf{Architecture-specific input preference}: While transformers benefit from Kalman fusion, CNN-Mamba performs better with raw input (89.98\% vs 88.09\%), suggesting CNNs are better suited for high-frequency raw signal processing.

    \item \textbf{Focal loss benefits imbalanced classification}: Focal loss improves recall by +6.41\% compared to BCE, which is critical for fall detection where missing falls has severe consequences.

    \item \textbf{Best configuration achieves 90.85\% F1}: The optimal configuration combines Kalman fusion, dual-stream transformer architecture with SE attention and temporal attention pooling, 50:50 embedding ratio, and focal loss, achieving 90.85\% F1 score on 21-fold LOSO-CV.
\end{enumerate}
