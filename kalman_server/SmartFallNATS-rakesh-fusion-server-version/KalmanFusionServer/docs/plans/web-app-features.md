# Web App Feature Roadmap

## Current Features (v0.1.0)

### Dashboard
- [x] Data file loading (JSON, CSV)
- [x] Config selection
- [x] Inference execution
- [x] Summary metrics display (F1, precision, recall, accuracy, specificity)
- [x] Confusion matrix heatmap
- [x] Probability distribution histogram
- [x] Adjustable threshold

### Model Comparison
- [x] Multi-config selection
- [x] Metrics comparison table
- [x] Bar chart comparison
- [x] Radar chart visualization
- [x] Best config identification

### Error Analysis
- [x] FN/FP filtering
- [x] Error list with sorting
- [x] Error details view
- [x] Errors by subject breakdown
- [x] Confidence distribution

### Signal Viewer
- [x] Class/error type filtering
- [x] UUID filtering
- [x] Interactive signal plots
- [x] SMV toggle
- [x] Raw data statistics
- [x] Multi-window comparison
- [x] Quick navigation
